# 概述
书名为《Maven实战》(Maven 3 in Action)

各章节的模块分开放在不同的目录中

和书中内容显著不一致的地方
- 书中使用了eclipse来开发项目，本项目使用IntelliJ IDEA
- 部分依赖和插件版本与书中不一致
- 书中<strong><em>maven-compiler-plugin</strong></em>的配置项为<strong><em>1.5</strong></em>，本项目使用<strong><em>1.8</strong></em>

所有模块代码都可以直接运行
